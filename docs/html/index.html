WHO PHSM Cleaning documentation¶



This library provides a workflow for harmonising non-pharmaceutical intervention data from seven existing providers. Currently implements cleaning of datasets from JH_HIT, CDC_ITF, OXCGRT, and ACAPS.

The WHO PHSM project is a collaboration between data providers, the World Health Organisation, and the London School of Hygiene and Tropical Medicine.

Download¶

The WHO PHSM dataset is released weekly.
